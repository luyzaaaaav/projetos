Diferenças entre Git e GitHub

A principal diferença é que o Git é um sistema de controle de versão que rastreia alterações em arquivos localmente, enquanto o GitHub é uma plataforma de hospedagem na web que usa o Git para armazenar, gerenciar e colaborar em projetos de código de forma remota e online. O Git é a ferramenta por trás do controle de histórico, enquanto o GitHub é um serviço que oferece um local centralizado para o trabalho com esses repositórios Git, proporcionando funcionalidades de colaboração e "redes sociais" de desenvolvedores.
