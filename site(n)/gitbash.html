Principais Comandos do GitBash

Após a instalação: git config --global user.name "Seu nome" git config --global user.email seu@email

Inicializar uma pasta com o git: git init

Adicinoar arquivos para gerar uma versão do git: git add nome_do_arquivo

Para gerar uma versão: git commit -m "Uma mensagem do que foi feito"

Verificar arquivos que foram alterados: git status

Verificar o log das modificações: git log
