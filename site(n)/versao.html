Como eu faço uma versão do projeto com o Git?

Para criar uma versão de um projeto com o Git, você precisa inicializar um repositório com git init ou clonar um existente, adicionar as alterações com git add, e então criar um commit (um "salvamento" do projeto naquele ponto) com git commit, adicionando uma mensagem descritiva. Para um projeto já existente e não versionado, o processo inclui cd para a pasta do projeto e depois git init para criar o repositório local, seguido pelos comandos add e commit.
